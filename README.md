# 🌍 HDI Prediction System using Machine Learning

## 📌 Project Overview

The HDI Prediction System is a Machine Learning-based web application that predicts the Human Development Index (HDI) using important socio-economic indicators. The system provides quick and reliable predictions through a simple and user-friendly web interface.

This project aims to demonstrate the application of Machine Learning in analyzing development indicators and supporting data-driven decision-making.

---

## 🚀 Features

- Predicts Human Development Index (HDI)
- User-friendly web interface
- Machine Learning-based prediction
- Fast and accurate results
- Flask backend integration
- Easy to deploy and maintain

## ⚙️ Installation

### Clone the Repository

```bash

```

### Navigate to Project

```bash

```

### Install Dependencies


```bash

```

## 📊 How It Works

1. User enters the required HDI indicators.
2. The input data is sent to the Flask backend.
3. The trained Machine Learning model processes the data.
4. The system predicts the Human Development Index.
5. The prediction result is displayed on the web page.
